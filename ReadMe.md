# Sentiment Analysis Microservice

A production-ready REST API for real-time sentiment analysis using a fine-tuned BERT model. Built with FastAPI, Hugging Face Transformers, and Docker.

## Features
- Modular Python design (separate modules for config, preprocessing, model, API)
- RESTful API with automatic OpenAPI documentation
- Version control with Git (GitHub repository)
- Containerized with Docker for easy deployment
- Unit tests and comprehensive documentation
